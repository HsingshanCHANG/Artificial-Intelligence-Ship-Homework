## 5 目录结构

| 文件（夹）名字          | 描述                                                            |
| -----------------| ------------------------------------ |
| data                         | 数据集                                                         |
| images                     | 存放图片（模型流程图）                                 |

| code                        | 模型代码（与“代码及处理结果.ipynb”内容一致） |

| ResultOfWPRDCNN.txt | 论文算法训练结果统计，TXT格式文件记录        |
| ResultOfWPRDCNN.csv| 论文算法训练结果统计，CSV格式文件记录        |
| 大作业要求.txt             | 课程要求（拷贝自QQ群公告栏）                      |
| README.md               | 说明、Help                                                  |
